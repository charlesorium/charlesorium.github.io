符号逻辑学基础1

Posted on 2019-07-23 In logic

about logic

​ 学符号逻辑有三条上升之路：哲学、计算机科学和数学。逻辑学作为哲学的分支，历史悠久，发展至今，出现了符号逻辑学。数学也有符号逻辑，但称作布尔代数。计算机科学从数学中独立出来后，布尔代数依旧影响着计科的发展，是计科的重要部分。

​ 这一系列笔记称为符号逻辑基础，意味着此笔记与哲学有很强的联系。

basic definitions

  1. 陈述句： $\forall\phi, \begin{cases}\phi是真的吗? \\ \phi是假的吗?\end{cases}$，有意义，则$\phi$为陈述句

  2. n元联结词

  3. 简单句与复合句

  4. 记法：

    | 采用 | 常见 | 波兰 |
    | ————————- | ———————————————— | —— |
    | $\sim$ | $\neg,\bar{A},\rightharpoondown$ | $N$ |
    | $\and$ | $\&$ | $K$ |
    | $\or$ | | $A$ |
    | $\rightarrow$ | $\supset,\Rightarrow$ | $C$ |
    | $\leftrightarrow$ | $\equiv,\Leftrightarrow$ | $E$ |

  5. $\mathcal{L}_0-$公式：$\mathcal{L}_0$的所有命题变号皆为$\mathcal{L}_0-$公式。若$\phi$是$\mathcal{L}_0-$公式，则$\sim\phi$也是$\mathcal{L}_0-$公式。若$\phi,\psi$皆为$\mathcal{L}_0-$公式，则$(\phi\and\psi),(\phi\or\psi),(\phi\rightarrow\psi),(\phi\leftrightarrow\psi)$都是$\mathcal{L}_0-$公式。只有这些是$\mathcal{L}_0-$公式。

  6. 重言蕴含（as a verb），重言后承（as a noun）：

    $\{\phi_1,\phi_2,\cdots,\phi _n \}$重言蕴含$\psi\phi_1,\phi_2,\cdots,\phi_n$的真值都是$_T$，那么$\psi$的真值也一定是$_T$

  7. 重言等值：互相重言蕴含

  8. 可满足的：$\Gamma$的联合真值表中，存在一行，在该行中的公式，真值都是$_T$
